# Phase 1 – Spec-Driven Todo In-Memory Console Application

## Project Overview
This project is a simple Python-based Todo application built as a console program.  
It follows **Spec-Driven Development**, as required by **Hackathon II**, ensuring clear scope, structure, and controlled implementation.

---

## Phase Information
- This repository contains **Phase 1 only**.
- Phase 1 focuses on a **single-user, in-memory Todo console application**.

---

## Features Implemented
- Add a task  
- View all tasks  
- Update a task  
- Delete a task  
- Mark a task as complete or incomplete  

---

## Technology Stack
- Python  
- Spec-Kit Plus  
- AI Coding Agent (Qwen / Gemini)  

---

## Spec-Driven Workflow
The project follows this spec-driven process:
- `sp.constitution`
- `sp.specify`
- `sp.plan`
- `sp.tasks`
- `sp.implement`
